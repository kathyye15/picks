## How to get started

Create a `.env.local` file containing your Google API key:

```
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY="your api key here"
```

Run the following:

```
npm i
npm run dev
```

Then visit http://localhost:3000/ to view your picks!
